Batch Normalization

Jun 21, 2020

https://cs231n.github.io/neural-networks-2/

Batch normalization is normalizing based on each feature dimention (axis = 0) cross all examples in the batch. While layer normalziation is for single training example.

Gradient calculation

It’s important to draw the computation dependency graph to better calculate the backward gradient.

The gradient calculation can be made simplified by using the graph and combining common terms in the expression.

One good gradient calculation post : https://kevinzakka.github.io/2016/09/14/batch_normalization/

Fully connected network impl

Number of layers = 1 + number of hidden layers

Backprop is kind of a reverse of forward.

The intermediate results of forward pass such as (x, w, b) are cached for later backward pass. Each layer can have such cache, then we will have a list of cache.

We can store parameters (gamma, beta, w, b etc.) in a map for each layer, indexed by (type + layer index) string.
